# print-shelver-3d
Print Shelver 3D es un robot que automatiza la extracción de piezas impresas en 3D y las coloca en estantes de almacenamiento.   Diseñado para la automatización del post-proceso en impresoras FDM, reduce la intervención manual, mejora la eficiencia del flujo de trabajo y habilita ciclos de impresión 24/7.
